Git 環境安裝

Git

簡介

Git 是一種版本控制系統，用於追蹤文件的變化，協同工作和協調代碼。

安裝

  1. 下載安裝檔 Git.

  1. 開啟安裝檔並選擇"Install"

  1. 完成安裝後，，可以在終端或命令提示字元中輸入 git --version 驗證安裝。

GitHub Desktop

簡介

GitHub Desktop 是 GitHub 官方提供的桌面應用程式，用於簡化 Git 操作流程。

安裝

  1. 下載 GitHub Desktop安裝檔

  1. 開啟安裝後，打開應用程式，登入 GitHub 帳號 (如沒有帳號請點此申請)。

使用 GitHub Desktop

  1. Clone 存儲庫：在 GitHub Desktop 中，點擊左上方 "File" -> "Clone Repository"，選擇要複製的存儲庫。

  1. 左上方 "Repository" -> "Show in Explorer"或"Open in Visual Studio Code"，即可開啟儲存庫並進行編輯。

  1. 對儲存庫進行變更後，在左下方輸入提交訊息，點擊 "Commit to main"。

  1. Push 變更：點擊 "Repository" -> "Push origin" 將變更推送到遠端儲存庫。

以上是 Git 和 GitHub Desktop 的基本安裝和使用方法。
